# EasyEdit
EasyEdit, an all-in-one smart document editor powered by natural language prompts



###  What Is EasyEdit?
EasyEdit is your all-in-one smart editor that helps you make instant changes to any document and this can be Google Docs, Sheets, PDFs, or Forms, all in one place.  
Upload a file, type what you want fixed, and EasyEdit instantly applies those edits without switching apps.



###  Why It Matters
We have all been there, needing to edit a document quickly but struggling with limited tools, multiple formats, or the wrong device.  
EasyEdit eliminates that frustration by offering one seamless platform where every document type feels easy to edit, even on mobile.

###  Key Features
-  **Universal File Editing:** Supports Google Docs, Sheets, PDFs, and Forms.  
-  **Prompt-Based Edits:** Simply say or type what you want changed.  
-  **Instant Formatting:** Adjust spacing, resize rows, fix grammar, or reformat layouts instantly.  
-  **Cloud Sync:** Connect with Google Drive or OneDrive for easy file access.  
-  **Cross-Platform Access:** Available on desktop, web, and mobile.  
-  **Secure Processing:** Files are encrypted and deleted after editing.  



###  Technologies Used
- **Frontend:** React.js, HTML5, TailwindCSS  
- **Backend:** Node.js, Express.js  
- **Database:** PostgreSQL  
- **Storage:** AWS S3 / Google Drive API  
- **Authentication:** Firebase Auth  
- **NLP Engine:** Python-based microservice for prompt understanding

To simplify document editing for everyone, anywhere, anytime, on any device.
